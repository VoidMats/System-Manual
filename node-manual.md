#### Deep clone
From node version 11, there is now **'structuredClone'**.
```
const oldFeature = {
  id: "660ec222107581c0395980ca",
  type: "Feature",
  properties: {
    attr_string: "test_string",
    attr_integer: 10,
  },
  geometry: {
    type: "Point",
    coordinates: [
      668000,
      6616000,
    ],
  }
}
const copyFeature = structuredClone(oldFeature);
oldFeature.properties.attr_string = "changed";

console.log(JSON.stringify(oldFeature) !== JSON.stringify(copyFeature))   // true
```
